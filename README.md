# World Cup Simulator ⚽

This project was developed as part of the **Katalyst Data Management Internship Selection Process – 2026**.

The application simulates a FIFA World Cup tournament by consuming an API with the 32 national teams, generating random groups, simulating matches, and progressing through the knockout stages until the final champion.

---

## 🚀 Technologies Used

- HTML
- CSS
- JavaScript
- Fetch API

---

## 📊 Features

- Fetch 32 national teams from the provided API
- Randomly generate 8 groups (A–H)
- Display all teams in their respective groups
- Simulate group stage matches
- Generate random match results
- Display all match results

---

## ⚙️ How It Works

1. The application fetches the teams from the API:
https://development-internship-api.geopostenergy.com/WorldCup/GetAllTeams

2. The teams are randomly shuffled and divided into **8 groups of 4 teams**.

3. Each group simulates **3 rounds of matches**.

4. Match results are randomly generated and displayed on the screen.
